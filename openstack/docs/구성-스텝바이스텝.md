## Step 1. OpenStack CLI 접속 + 상태 점검

### 1-1. MobaXterm → OpenStack 호스트

```
Host: 172.16.8.100
User: root
```

### 1-2. kolla_toolbox 진입

```bash
docker exec -u 0 -it kolla_toolbox bash
```

### 1-3. admin-openrc (없으면 한 번만)

```bash
# toolbox 밖(호스트)에서:
docker cp /etc/kolla/admin-openrc.sh kolla_toolbox:/tmp/admin-openrc.sh

# toolbox 안에서:
source /tmp/admin-openrc.sh
export OS_AUTH_URL=http://172.16.8.100:5000
```

### 1-4. 현재 상태 확인 (복사해서 실행)

```bash
openstack server list
openstack network list
openstack router list
openstack keypair list
openstack floating ip list
openstack compute service list
```

**기록할 것:** server Status, FIP 연결 여부

## Step 3. SSH Keypair

```bash
openstack keypair list
openstack keypair show project_key
```

없으면 (호스트에서 pub 파일 경로 맞게):

```bash
openstack keypair create --public-key @/path/to/project_key.pub project_key
```

## Step 4. tenant VM 켜기 + FIP

### Horizon (GUI)

1. http://172.16.8.100 → admin 로그인
2. Project → Compute → Instances
3. SHUTOFF → **Start Instance**
4. Network → Floating IPs → master에 `172.16.8.146` 연결

### CLI

```bash
# SHUTOFF VM 전부 시작
for id in $(openstack server list -f value -c ID -c Status | awk '$2=="SHUTOFF"{print $1}'); do
  openstack server start "$id"
done

# FIP 연결 (이미 있으면 생략)
openstack server add floating ip master 172.16.8.146
```
