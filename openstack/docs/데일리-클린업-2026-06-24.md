- Ubuntu 24.04 root SSH / `ssh-copy-id` 실패 → `sshd_config.d/99-root-login.conf` 추가 필요
- `globals.yml` 편집 시 `nano` 명령어가 파일에 섞여 YAML syntax error
- NIC 이름 강의(`ens32`/`ens33`)와 실제(`ens33`/`ens34`) 불일치 → globals/netplan 수동 조정
- `openstack network list` 비어 있음 → Kolla deploy 후 **리소스 직접 생성**이 정상 (오해)

- control 호스트 `virsh` 없음 → **`docker exec nova_libvirt virsh list --all`** 로 확인
- compute hostname `compute1` (강의 `compute`와 명칭만 상이)
