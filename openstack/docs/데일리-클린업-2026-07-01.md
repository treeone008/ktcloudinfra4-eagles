## 5. 내일 FIP 트러블슈팅 — 빠른 명령

```bash
source /root/venv/bin/activate && source /etc/kolla/admin-openrc.sh

# 상태 확인
openstack floating ip list -f table
openstack server show swarm-mg-02 -c status -c addresses -f table
openstack router show project-router -c status -c external_gateway_info
openstack network agent list | grep -i l3

# control 경로 비교 (219 OK / 243 FAIL 패턴)
ip route get 172.16.8.219
ip route get 172.16.8.243
ping -c2 172.16.8.219
ping -c2 172.16.8.243

# VM 부팅·sshd
openstack console log show swarm-mg-02 | tail -30
ssh -i /root/.ssh/id_rsa ubuntu@172.16.8.243 "hostname"

# L3 agent 재시작 (network 노드)
ssh root@172.16.8.101 "docker restart \$(docker ps --format '{{.Names}}' | grep neutron_l3)"
```
